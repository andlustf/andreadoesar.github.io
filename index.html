Augmented Reality Simon Says

App Description

For my midterm project, I have decided to take the classic game of Simon Says and turn it into Augmented Reality. The idea is, you'll be able to play a fun and interactive game, while stimulating your brain, and because it is in AR, you will be able to play it as long as you have access to the technology, no need for external devices that occupy a lot of space.

Storyboard

  1. User initiates the application
  2. User shows the marker to initiate AR application
  3. User then presses start button to begin the game
  4. User follows along the game as the game progresses

AR Features

  • Marker recognition
  • UI buttons
  • UI canvas group
